📚 Типы данных с плавающей точкой

В C++ есть три типа для работы с дробными числами: float, double и long double. Каждый тип имеет разную точность и занимает разное количество памяти.

C++ 
#include <iostream>
#include <limits>
#include <iomanip>

int main() {
    float f{3.14159f};
    double d{3.141592653589793};
    long double ld{3.141592653589793238L};
    
    std::cout << std::setprecision(15);
    std::cout << "float: " << f << '\n';
    std::cout << "double: " << d << '\n';
    std::cout << "long double: " << ld << '\n';
    
    // Информация о типах
    std::cout << "\nХарактеристики типов:\n";
    std::cout << "float размер: " << sizeof(float) << " байт\n";
    std::cout << "float точность: " << std::numeric_limits<float>::digits10 << " цифр\n";
    std::cout << "double размер: " << sizeof(double) << " байт\n";
    std::cout << "double точность: " << std::numeric_limits<double>::digits10 << " цифр\n";
    
    return 0;
}
Компиляция успешна! ✅
float: 3.14159011840820
double: 3.14159265358979
long double: 3.14159265358979
Характеристики типов:
float размер: 4 байт
float точность: 6 цифр
double размер: 8 байт
double точность: 15 цифр
