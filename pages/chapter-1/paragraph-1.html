§ 1. Фундаментальные типы данных

В программировании тип данных определяет, какие значения может хранить переменная и какие операции можно с ней выполнять. C++ — это строго типизированный язык, что означает, что каждая переменная должна иметь определённый тип, который известен на этапе компиляции.

Что такое фундаментальные типы

Фундаментальные типы данных (fundamental types) — это встроенные типы языка C++, которые представляют базовые значения: целые числа, числа с плавающей точкой, символы и логические значения. Эти типы являются основой для построения более сложных типов данных.

Фундаментальные типы делятся на несколько категорий:

  • Целочисленные типы (int, short, long, long long)
  • Типы с плавающей точкой (float, double, long double)
  • Символьные типы (char, wchar_t, char16_t, char32_t)
  • Логический тип (bool)
  • Тип void (отсутствие типа)

Целочисленные типы

Целочисленные типы используются для хранения целых чисел (без дробной части). В C++ существует несколько целочисленных типов, различающихся размером и диапазоном значений.

[basic.fundamental] §6.7.1
There are five standard signed integer types: "signed char", "short int", "int", "long int", and "long long int".

long long — Длинное длинное целое

Тип long long (введён в C++11) предназначен для хранения очень больших целых чисел. Гарантированно имеет размер минимум 64 бита, что позволяет работать с числами до 9 квинтиллионов.

Размер 8 байт (64 бита)
Диапазон (signed) $-9.2 \times 10^{18}$ до $9.2 \times 10^{18}$
Применение Большие числа, ID, время
C++ 
#include <iostream>
#include <limits>

int main() {
    long long bigNumber{9223372036854775807LL};
    unsigned long long evenBigger{18446744073709551615ULL};
    
    std::cout << "long long: " << bigNumber << '\n';
    std::cout << "unsigned long long: " << evenBigger << '\n';
    
    std::cout << "\nХарактеристики long long:\n";
    std::cout << "Размер: " << sizeof(long long) << " байт\n";
    std::cout << "Минимум: " << std::numeric_limits<long long>::min() << '\n';
    std::cout << "Максимум: " << std::numeric_limits<long long>::max() << '\n';
    
    return 0;
}

long — Длинное целое

Тип long предназначен для хранения больших целых чисел. На 32-битных системах обычно 32 бита, на 64-битных может быть 64 бита (зависит от платформы).

Размер 4-8 байт (32-64 бита)
Диапазон (32-bit) $-2.1 \times 10^9$ до $2.1 \times 10^9$
Применение Большие счётчики, индексы
C++ 
#include <iostream>
#include <limits>

int main() {
    long distance{2147483647L};
    unsigned long population{4294967295UL};
    
    std::cout << "long: " << distance << '\n';
    std::cout << "unsigned long: " << population << '\n';
    
    std::cout << "\nХарактеристики long:\n";
    std::cout << "Размер: " << sizeof(long) << " байт\n";
    std::cout << "Минимум: " << std::numeric_limits<long>::min() << '\n';
    std::cout << "Максимум: " << std::numeric_limits<long>::max() << '\n';
    
    return 0;
}

int — Целое число

Тип int — это наиболее часто используемый целочисленный тип. Обычно имеет размер 32 бита и является оптимальным выбором для большинства целочисленных вычислений.

Размер 4 байта (32 бита)
Диапазон $-2.1 \times 10^9$ до $2.1 \times 10^9$
Применение Общие вычисления, счётчики
C++ 
#include <iostream>
#include <limits>

int main() {
    int age{25};
    int temperature{-15};
    unsigned int count{1000};
    
    std::cout << "Возраст: " << age << '\n';
    std::cout << "Температура: " << temperature << '\n';
    std::cout << "Счётчик: " << count << '\n';
    
    std::cout << "\nХарактеристики int:\n";
    std::cout << "Размер: " << sizeof(int) << " байт\n";
    std::cout << "Минимум: " << std::numeric_limits<int>::min() << '\n';
    std::cout << "Максимум: " << std::numeric_limits<int>::max() << '\n';
    
    return 0;
}

short — Короткое целое

Тип short используется для экономии памяти при работе с небольшими целыми числами. Имеет размер 16 бит и может хранить значения от -32,768 до 32,767.

Размер 2 байта (16 бит)
Диапазон $-32'768$ до $32'767$
Применение Экономия памяти, малые числа
C++ 
#include <iostream>
#include <limits>

int main() {
    short year{2024};
    short score{-100};
    unsigned short port{8080};
    
    std::cout << "Год: " << year << '\n';
    std::cout << "Счёт: " << score << '\n';
    std::cout << "Порт: " << port << '\n';
    
    std::cout << "\nХарактеристики short:\n";
    std::cout << "Размер: " << sizeof(short) << " байт\n";
    std::cout << "Минимум: " << std::numeric_limits<short>::min() << '\n';
    std::cout << "Максимум: " << std::numeric_limits<short>::max() << '\n';
    
    return 0;
}

Модификаторы signed и unsigned изменяют диапазон значений. unsigned int хранит только положительные числа ($0$ до $4'294'967'295$), удваивая положительный диапазон за счёт отсутствия отрицательных значений.

🔢

short (short int)

Минимум 16 бит. Диапазон: от $-32'768$ до $32'767$. Используется для экономии памяти при работе с небольшими числами.

🔢

int

Обычно 32 бита. Диапазон: от $-2'147'483'648$ до $2'147'483'647$. Наиболее часто используемый целочисленный тип.

🔢

long (long int)

Минимум 32 бита. На 64-битных системах часто 64 бита. Для больших целых чисел.

🔢

long long

Минимум 64 бита. Диапазон: от $-9'223'372'036'854'775'808$ до $9'223'372'036'854'775'807$. Для очень больших чисел (C++11).

Типы с плавающей точкой

Типы с плавающей точкой используются для хранения вещественных чисел (с дробной частью). Они представляют числа в формате с плавающей точкой согласно стандарту IEEE 754.

📊

float

Одинарная точность, обычно 32 бита. Точность: ~7 десятичных цифр. Диапазон: $\pm 3.4 \times 10^{38}$.

📊

double

Двойная точность, обычно 64 бита. Точность: ~15 десятичных цифр. Диапазон: $\pm 1.7 \times 10^{308}$. Стандартный выбор для вещественных чисел.

📊

long double

Расширенная точность, минимум 64 бита (часто 80 или 128 бит). Максимальная точность для вещественных вычислений.

Пример: Типы с плавающей точкой

C++ 
#include <iostream>
#include <limits>
#include <iomanip>

int main() {
    float f{3.14159f};
    double d{3.141592653589793};
    long double ld{3.141592653589793238L};
    
    std::cout << std::setprecision(15);
    std::cout << "float: " << f << '\n';
    std::cout << "double: " << d << '\n';
    std::cout << "long double: " << ld << '\n';
    
    // Информация о типах
    std::cout << "\nХарактеристики типов:\n";
    std::cout << "float размер: " << sizeof(float) << " байт\n";
    std::cout << "float точность: " << std::numeric_limits<float>::digits10 << " цифр\n";
    std::cout << "double размер: " << sizeof(double) << " байт\n";
    std::cout << "double точность: " << std::numeric_limits<double>::digits10 << " цифр\n";
    
    return 0;
}

Символьные типы

Символьные типы используются для хранения символов. Тип char обычно занимает 1 байт и может хранить символы ASCII. Для Unicode существуют специальные типы.

C++ 
#include <iostream>

int main() {
    char c{'A'};
    wchar_t wc{L'Я'};
    char16_t c16{u'€'};
    char32_t c32{U'🔥'};
    
    std::cout << "char: " << c << '\n';
    std::cout << "char размер: " << sizeof(char) << " байт\n";
    std::cout << "wchar_t размер: " << sizeof(wchar_t) << " байт\n";
    std::cout << "char16_t размер: " << sizeof(char16_t) << " байт\n";
    std::cout << "char32_t размер: " << sizeof(char32_t) << " байт\n";
    
    return 0;
}

Логический тип bool

Тип bool используется для хранения логических значений: true (истина) или false (ложь). Это фундаментальный тип для условных выражений и логических операций.

C++ 
#include <iostream>

int main() {
    bool isActive{true};
    bool isComplete{false};
    
    std::cout << "isActive: " << isActive << '\n';
    std::cout << "isComplete: " << isComplete << '\n';
    std::cout << std::boolalpha;  // Вывод true/false вместо 1/0
    std::cout << "isActive: " << isActive << '\n';
    std::cout << "bool размер: " << sizeof(bool) << " байт\n";
    
    return 0;
}

Использование std::numeric_limits

Шаблон класса std::numeric_limits из заголовка <limits> предоставляет информацию о свойствах арифметических типов.

C++ 
#include <iostream>
#include <limits>

int main() {
    std::cout << "Информация о типе int:\n";
    std::cout << "Минимум: " << std::numeric_limits<int>::min() << '\n';
    std::cout << "Максимум: " << std::numeric_limits<int>::max() << '\n';
    std::cout << "Знаковый: " << std::numeric_limits<int>::is_signed << '\n';
    std::cout << "Целочисленный: " << std::numeric_limits<int>::is_integer << '\n';
    
    std::cout << "\nИнформация о типе double:\n";
    std::cout << "Минимум: " << std::numeric_limits<double>::min() << '\n';
    std::cout << "Максимум: " << std::numeric_limits<double>::max() << '\n';
    std::cout << "Точность (цифр): " << std::numeric_limits<double>::digits10 << '\n';
    std::cout << "Эпсилон: " << std::numeric_limits<double>::epsilon() << '\n';
    
    return 0;
}

Итоги параграфа

Что мы узнали:

  • Фундаментальные типы — это встроенные типы C++, представляющие базовые значения.
  • Целочисленные типы (short, int, long, long long) используются для целых чисел.
  • Типы с плавающей точкой (float, double, long double) для вещественных чисел.
  • Символьные типы (char, wchar_t, char16_t, char32_t) для символов и текста.
  • Логический тип bool для хранения true/false.
  • std::numeric_limits предоставляет информацию о свойствах типов.
