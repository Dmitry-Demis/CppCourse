    Литералы, операторы и выражения

    Три фундаментальных строительных блока любого вычисления в C++.

    1. Литералы

    Литерал — фиксированное значение, записанное прямо в коде. Компилятор знает его тип по синтаксису.

    ?? Виды литералов
    • 42 — целочисленный (int)
    • 42L — целочисленный (long)
    • 3.14 — с плавающей точкой (double)
    • 3.14f — с плавающей точкой (float)
    • 'A' — символьный (char)
    • "hello" — строковый (const char*)
    • true / false — булев (bool)

    2. Операторы

    Оператор выполняет операцию над одним или несколькими операндами и возвращает результат.

    C++ — литералы и операторы 
    // загрузка...

    3. Выражения

    Выражение — комбинация литералов, переменных и операторов, которая вычисляется в единственное значение с определённым типом.

    C++ — выражения 
    // загрузка...
    ?? Инструкция vs выражение
    • Выражение вычисляется в значение: x + 5
    • Инструкция выполняет действие и заканчивается ;: x = x + 5;
    • Выражение + ; = инструкция-выражение (expression statement)

    4. Приоритет операторов

    Как в математике: * и / выполняются раньше + и -. Скобки () меняют порядок вычисления. При сомнениях — используйте скобки явно.
