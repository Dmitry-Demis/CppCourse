    Переменные и типы данных

    Программы работают с данными. Чтобы хранить данные в памяти, используются переменные — именованные области памяти.

    Объекты и переменные

    В C++ объект — это область памяти с типом и значением. Переменная — это именованный объект. Имя позволяет обращаться к объекту в коде.

    Основные типы данных

    Тип определяет, сколько памяти занимает переменная и какие операции над ней допустимы.

    ?? Базовые типы C++
    • int — целое число (обычно 4 байта): -2 147 483 648 до 2 147 483 647
    • double — число с плавающей точкой (8 байт)
    • char — один символ (1 байт)
    • bool — логическое значение: true или false
    • std::string — строка текста (из библиотеки <string>)

    Объявление переменной

    Синтаксис: тип имя; или тип имя = значение; Объявление сообщает компилятору о существовании переменной и резервирует память.

    ?? Одно определение

    Переменную нельзя объявить дважды в одной области видимости. int x; int x; — ошибка компиляции.
