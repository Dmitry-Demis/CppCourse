    Комментарии

    Комментарий — это текст в коде, который компилятор полностью игнорирует. Комментарии пишутся для людей: для себя в будущем и для коллег.

    1. Однострочный комментарий

    Начинается с // и продолжается до конца строки. Это самый распространённый вид комментариев в C++.

    C++ — виды комментариев 
    // загрузка...

    2. Многострочный комментарий

    Начинается с /* и заканчивается */. Может занимать несколько строк. Вложенные многострочные комментарии не поддерживаются.

    ⚠️ Нельзя вкладывать

    /* /* вложенный */ */ — ошибка компиляции. Первый */ закрывает весь комментарий, а второй */ — лишний.

    3. Что комментировать

    ✅ Хорошие комментарии
    • Объясняют зачем написан код, а не что он делает
    • Описывают неочевидные решения и их причины
    • Предупреждают о подводных камнях
    • Документируют параметры и возвращаемые значения функций
    ❌ Плохие комментарии
    • x = x + 1; // увеличиваем x на 1 — очевидно из кода
    • Устаревшие комментарии, которые не соответствуют коду
    • Закомментированный мёртвый код без объяснения
