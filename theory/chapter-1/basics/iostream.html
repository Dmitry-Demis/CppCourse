    Ввод и вывод: iostream

    Библиотека <iostream> предоставляет инструменты для взаимодействия с пользователем через консоль. iostream = input/output stream.

    1. std::cout — вывод

    std::cout (character output) выводит данные в стандартный поток вывода (консоль). Оператор << передаёт данные в поток. Можно цеплять несколько значений подряд.

    2. Перевод строки: \n vs endl

    💡 \n предпочтительнее endl
    • "\n" — просто символ новой строки, быстро
    • std::endl — новая строка + принудительный сброс буфера (flush), медленнее
    • Используйте "\n" в большинстве случаев
    • std::endl нужен только когда важно немедленно вывести данные

    3. std::cin — ввод

    std::cin (character input) читает данные из стандартного потока ввода (клавиатура). Оператор >> извлекает данные из потока и записывает в переменную.

    ⚠️ cin и пробелы

    std::cin >> x пропускает пробелы и читает до следующего пробела/переноса. Для чтения строки с пробелами используйте std::getline(std::cin, str).

    4. Зачем std::

    std:: — это пространство имён (namespace) стандартной библиотеки. Оно предотвращает конфликты имён. Можно написать using namespace std;, но в больших проектах это считается плохой практикой.
