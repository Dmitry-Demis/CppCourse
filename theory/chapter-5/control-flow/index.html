    Глава V

    Управление потоком

    До этой главы программы выполнялись строго сверху вниз — одинаково при каждом запуске. Операторы управления потоком меняют это: программа может ветвиться, повторять блоки кода и реагировать на данные времени выполнения. Именно здесь начинается настоящее программирование.

    §5.1
    Введение в управление потоком
    Путь выполнения, ветвление, категории операторов: условия, циклы, прыжки, исключения
    §5.2
    Оператор if и блоки
    Синтаксис if/else, блоки { }, неявные блоки, if-else vs if-if, область видимости
    §5.3
    Типичные ошибки с if
    Dangling else, уплощение вложенных if, null statement, = вместо ==
