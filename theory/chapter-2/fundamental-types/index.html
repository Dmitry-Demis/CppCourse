Фундаментальные типы данных

В программировании тип данных определяет, какие значения может хранить переменная и какие операции можно с ней выполнять. C++ — это строго типизированный язык, что означает, что каждая переменная должна иметь определённый тип, который известен на этапе компиляции.

Что такое фундаментальные типы

Фундаментальные типы данных (fundamental types) — это встроенные типы языка C++, которые представляют базовые значения: целые числа, числа с плавающей точкой, символы и логические значения. Эти типы являются основой для построения более сложных типов данных.

Фундаментальные типы делятся на несколько категорий:

🔵

Тип void

Пустой тип, nullptr
🔢

Целочисленные типы

int, long, short, long long
📊

Типы с плавающей точкой

float, double, long double
🔤

Символьные типы

char, wchar_t, char16_t, char32_t
✅

Логический тип

bool (true/false)
